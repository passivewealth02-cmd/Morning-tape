**Primary (recommended) — 139 chars:**

```

```

**Alternate A — checklist-forward, 138 chars:**

```

```

**Alternate B — luxury/planner angle, 137 chars:**

```
Luxury Wedding Planner Spreadsheet | All-in-One Excel Google Sheets Budget, Guest List, Vendor, RSVP & Seating Chart Planner Template
```

**Title strategy**
- Front-load the #1 keyword phrase **"Wedding Planner Spreadsheet"**
  (Etsy weights the first ~40 chars most heavily).
- Stack the highest-volume long-tail terms buyers actually search:
  *wedding budget*, *guest list*, *RSVP tracker*, *seating chart*,
  *vendor tracker*, *Excel*, *Google Sheets*, *template*.
- Use `|` separators (Etsy reads each segment as a keyword phrase).
- No ALL-CAPS keyword stuffing — Etsy 2024+ favors readable titles.

## 5. CONVERSION & RANK BOOSTERS (operational)

- **Price anchoring:** show the $49 bundle next to the $39 single — the
  bundle reads as the obvious value. Consider a "launch" sale (40% off)
  for the first 2 weeks to spike sales velocity (Etsy's strongest rank
  signal for new listings).
- **First 10 photos:** use the generated set (hero first). Etsy rewards
  listings that use all 10 image slots + a video.
- **Add a video:** a 5–10s screen-scroll of the dashboard updating ranks
  higher and lifts conversion ~yes measurably.
- **Renew/refresh** the listing every few weeks early on; keep the title
  stable once it ranks (don't churn keywords after you find a winner).
- **Reviews flywheel:** instant digital delivery + a "message me if
  anything's off" line drives 5-star reviews, which compound rank.
- **Sections/shop:** put it in a "Wedding Planners" shop section named
  with the keyword; cross-link your other Command Center products.
- **Seasonality:** wedding search peaks Dec–Feb (engagement season) and
  again May–Aug. Time promos accordingly.
