## Signature automation

| Metric | How it's computed |
| ------ | ----------------- |
| Days Until Wedding | `=MAX(WeddingDate-TODAY(),0)` |
| Wedding Readiness Score | weighted average of 6 health dimensions |
| RSVP % / Attendance % | `COUNTIF` + `SUMPRODUCT` over Guest CRM |
| Tables Filled | accepted seats ÷ seats-per-table |
| Payments Outstanding | `SUMPRODUCT((PayPaid<>"Yes")*Balance)` |
| Planning Completion | `COUNTIF(ChkDone,"Yes")/COUNTA(ChkTask)` |
| Seating capacity | per-table open seats + over-capacity flag |
| Meal counts | per-meal accepted-seat totals |

43 named ranges keep the whole system wired and make the Google Sheets
edition a 1-to-1 mirror.
